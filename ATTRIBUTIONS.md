# Carwash Management System

A modern carwash management system designed for efficient business operations, customer management, scheduling, analytics, and point-of-sale transactions.

## Features

- Modern glassmorphism UI
- Interactive dashboard analytics
- Point of Sale (POS) system
- Customer management
- Scheduling and ticket management
- Inventory management
- Transaction history
- Excel report generation
- Responsive design

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- React Query
- Recharts
- Lucide React
- SQLite

## Installation

Clone the repository:

```bash
git clone https://github.com/ninyorico/carwash-.git
```

Install dependencies:

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deployment

Frontend is deployed using Vercel.
